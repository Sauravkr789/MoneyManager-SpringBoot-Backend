# Money-Manager-Backend

## About the Backend
This repository serves as the core engine for the Money Manager platform. It manages the business logic, data persistence, and API endpoints required to power a seamless financial tracking experience. The architecture focuses on scalability, clean code, and efficient database management using the Spring Framework.

## Technical Highlights

- **Full CRUD Functionality:** Implemented a complete data lifecycle for financial transactions, allowing the frontend to Create, Read, Update, and Delete records with guaranteed data integrity.
- **RESTful API Development:** Designed and documented a series of endpoints to handle asynchronous requests from the React frontend.
- **Data Persistence:** Utilizes Hibernate (JPA) to map Java objects to a PostgreSQL relational database, ensuring efficient query execution and schema management.
- **Logic Optimization:** Features specialized service layers to handle complex financial calculations and data validation before persistence.

## Backend Tech Stack
- **Language:** Java
- **Framework:** Spring Boot, Hibernate (JPA)
- **API:** RESTful Web Services
- **Database:** PostgreSQL / MySQL
- **Tools:** Maven, Postman, IntelliJ IDEA
